# **Wiki ID**
### Web-encyclopedia with personalized access to pages

### Features:

* Creation, browsing, editing, deletion [wiki-pages](https://ru.wikipedia.org/wiki/%D0%92%D0%B8%D0%BA%D0%B8)
* Gallery on each page for media content
* Access levels - edit, view and public
* Notifications 

### Technical requirements:

* ASP.NET Core 3.1:sunglasses:

* MVC pattern :godmode:

* Using `Entity Framework` as ORM library :satisfied:

* User verification by email :love_letter:
